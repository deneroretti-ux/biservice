BI Service Site SaaS Final

Como instalar:
1. Apague a pasta atual: app/bi-service
2. Copie a pasta app/bi-service deste zip para dentro do seu projeto.
3. Copie os arquivos da pasta public deste zip para a pasta public do seu projeto.
4. Rode: npm run dev
5. Acesse: http://localhost:3000/bi-service

Publicar na Vercel:
1. Suba o projeto para o GitHub.
2. Entre em https://vercel.com
3. Clique em Add New > Project.
4. Importe o repositório.
5. Framework: Next.js.
6. Clique Deploy.
7. Depois de publicado, acesse: https://seu-projeto.vercel.app/bi-service
