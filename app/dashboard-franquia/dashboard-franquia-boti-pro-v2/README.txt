Dashboard Franquia Pro - BI Service

Como usar:
1. Copie a pasta app/dashboard-franquia-pro para a raiz do seu projeto Next.js.
2. Rode npm run dev.
3. Acesse /dashboard-franquia-pro.

Esta versão tem menu lateral por KPI:
Ticket médio, PA, meta, sellout, DRE, margem, ruptura, consultores, campanhas, evolução mensal, comparativos entre lojas, ranking e metas por vendedor.
